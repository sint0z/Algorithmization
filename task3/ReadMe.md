# # Лабораторная работа: Тестирование алгоритмов сортировки
**Вариант 25**  
**Сравнение пузырьковой сортировки, сортировки расчёской и пирамидальной сортировки**

---

## 1. Общие сведения

Данный проект предназначен для проведения сравнительного анализа трёх алгоритмов сортировки по скорости выполнения на различных типах массивов данных.

### Реализованные алгоритмы:
- **Пузырьковая сортировка (Bubble Sort)** — простейший алгоритм с квадратичной сложностью.
- **Сортировка расчёской (Comb Sort)** — улучшенная версия пузырьковой сортировки, сокращающая количество сравнений.
- **Пирамидальная сортировка (Heap Sort)** — эффективный алгоритм, основанный на структуре двоичной кучи, с асимптотической сложностью O(n log n).

---

## 2. Требования к тестированию

### Основные задачи:
- Реализовать минимум три алгоритма сортировки из указанного варианта.
- Провести тестирование каждого алгоритма на разных массивах.
- Замерить время выполнения (в наносекундах или миллисекундах).
- Провести минимум **5 запусков** каждого теста и учитывать **лучшее** или **среднее** значение.
- Подготовить **сравнительный анализ** результатов.
---

## 4. Метрики и методика измерений

### Основная метрика:
- **Время выполнения** — замеряется в миллисекундах или наносекундах.
