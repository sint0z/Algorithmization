# Лабораторная работа: Бинарные деревья на Java

## Описание
В этом проекте реализована структура узла и основные операции с бинарными деревьями на языке Java. Включены:
- Базовые обходы дерева: прямой (pre-order), центрированный (in-order), обратный (post-order), по уровням (level-order).
- Вычисление высоты дерева.
- Построение полного бинарного дерева (Full Binary Tree) с проверкой `isFull()`.
- Построение бинарного дерева поиска (BST) из массива целых чисел.
- Построение сбалансированного бинарного дерева из отсортированного массива.
- Вставка и удаление узлов по стратегии обхода по уровням (level order).

## Структура проекта
- `TreeNode` — класс для узла дерева (целое значение и ссылки на потомков).
- `TreeTraversal` — интерфейс с методами обхода и вычисления высоты.
- `TreeOperations` — интерфейс для вставки и удаления узлов.
- `TreeUtils` — реализация обходов и высоты.
- `FullBinaryTreeUtils` — проверка полного бинарного дерева и создание примера дерева.
- `BinarySearchTree` — класс для BST с вставкой и построением из массива.
- `BalancedTreeBuilder` — построение сбалансированного дерева из отсортированного массива.
- `BinaryTree` — дерево с вставкой/удалением по уровню.

## Как использовать
1. В классе `Main` демонстрируется:
    - Создание и обходы обычного дерева с вставкой по уровням.
    - Построение полного дерева и проверка `isFull()`.
    - Создание BST из массива и центрированный обход.
    - Построение сбалансированного дерева из отсортированного массива и обход по уровням.
2. Запустите `Main.java` для просмотра вывода и проверки работы методов.
