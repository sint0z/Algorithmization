# Репозиторий студента

## О проекте
Этот репозиторий создан для хранения лабораторных работ и заданий по курсу "Алгоритмизация и программирование" для 2-го курса обучения. Все работы выполнены студентом Стоцким Никитой Алексеевичем

## Содержание
В репозитории представлены решения практических задач по различным темам программирования и алгоритмов, включая структуры данных, сортировки, деревья и другие ключевые разделы курса.

## Цель
Главная цель — систематизация изученного материала и создание удобной базы примеров для дальнейшего обучения и повторения тем.

## Структура
- Каждый проект содержит документацию, исходный код и комментарии, чтобы облегчить понимание.
- Работы упорядочены по номерам и темам лабораторных.

## Использование
Репозиторий служит как личный учебный ресурс и может быть полезен студентам, изучающим аналогичные темы.

## Инструменты и окружение

- **Язык программирования:** Java
- **Среда разработки:** IntelliJ IDEA / Visual Studio Code
- **Система сборки:** Maven
- **Версия JDK:** 17+
- **Поддержка документирования:** Perplexity AI
