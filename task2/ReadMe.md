# # Лабораторная работа: Реализация связных списков на Java

## Обзор проекта

Проект реализует три типа связных списков с полным набором операций:
- **Односвязный список** (`SingleLinkedList<T>`)
- **Двусвязный список** (`DoubleLinkedList<T>`)
- **Циклический односвязный список** (`CircularLinkedList<T>`)

Все реализации:
- Поддерживают итерацию (`Iterable<T>`)
- Имеют корректное освобождение памяти (`clear()`)
- Валидируют индексы и граничные случаи
- Имеют Javadoc-документацию

## Особенности реализации

✅ **Освобождение памяти**: `clear()` обнуляет все ссылки (`item`, `next`, `prev`)  
✅ **Итераторы**: работают корректно, не зацикливаются  
✅ **Граничные случаи**: пустой список, один элемент, индексы  
✅ **Цикличность**: `tail.next = head`, `rotate()` сдвигает логически  
✅ **Разбиение**: `splitIntoTwo()` создаёт два полноценных цикла  
✅ **Javadoc**: полная документация методов и классов
